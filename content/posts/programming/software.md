## Declarative vs imperative language
Declarative programming (宣告式)
- **What**
- 代表為 functional programming
- (function 之間不會互相共用狀態)
- 程式碼通常是可以直接看到在做什麼 (重視 what)
 
Imperative programming (命令式)
- **how to do**
- 代表為 OOP 
- (狀態互相依賴)
- 看得見邏輯運算過程 (how to do)
