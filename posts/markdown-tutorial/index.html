斜體字

用底線 (underscore) 把要轉斜體字的字串夾起來 _TARGET_

結果: TARGET

粗體字

用兩個星星符號 (asterisk) 把要轉斜體字的字串夾起來 **TARGET**

結果: TARGET

標題

標題分成六種大小 在標題文字前面加上井字號 (hash mark)

井字號越少代表越大

# TARGET
## TARGET
### TARGET
#### TARGET
##### TARGET
###### TARGET

或是加上 - 或 = 也可以表示標題

連結

連結的格式為中括號內放文字，小括號內放連結 [TARGET DESCRIPTION](TARGET LINK)

結果: TARGET DESCRIPTION

無序條列

利用 + 或 - 或 * 來做無序條列

- TARGET1
- TARGET2

  second line
- TARGET3

  • TARGET1

  • TARGET2

    second line

  • TARGET3

有序條列

利用數字和點來做有序條列

1. TARGET1
2. TARGET2
3. TARGET3
  1. TARGET1
  2. TARGET2
  3. TARGET3

參考類型的連結

利用中括號來完成

This is [ref1][1] and [ref2][ref-2]

[1]: http://google.com/        "Google"
[ref-2]: http://search.yahoo.com/  "Yahoo Search"

This is ref1 and ref2
